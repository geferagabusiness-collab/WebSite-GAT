# SYSTEM OVERVIEW — WEBSITE GRUPO AXM TECHNOLOGY

## Objetivo General

Diseñar y desarrollar una plataforma web corporativa premium para Grupo AXM Technology orientada a:

- atraer nuevos clientes constantemente,
- transmitir autoridad tecnológica,
- demostrar capacidades reales en IA y automatización,
- servir como portal operativo empresarial,
- integrar el ecosistema Alexandro,
- y evolucionar posteriormente hacia una plataforma SaaS empresarial.

---

# Filosofía del Proyecto

El sitio web NO debe sentirse como una página corporativa tradicional.

Debe sentirse como:

- una empresa tecnológica avanzada,
- una experiencia interactiva,
- una plataforma viva impulsada por IA,
- una demostración de capacidades reales.

La experiencia visual debe reflejar:

- inteligencia artificial,
- automatización,
- premium tech,
- infraestructura avanzada,
- seguridad,
- innovación.

# UI UX RULES

# Regla Principal

Cada sección debe parecer una experiencia premium.

---

# Espaciado

- aire visual,
- separación limpia,
- diseño elegante.

---

# Motion

- microinteracciones suaves,
- animaciones fluidas,
- evitar exceso.

---

# Responsive

Prioridad:

1. Desktop premium.
2. Mobile optimizado.
3. Tablet.

---

# Interacción

Cada interacción debe sentirse:

- rápida,
- moderna,
- elegante.
